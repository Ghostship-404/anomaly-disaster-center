OVERVIEW

概述

ANO-012 的外形为一部产于2010年的诺基亚品牌智能按键手机，通体橘红色，机壳表面无明显磨损。机身顶端系有一根异常冗长的红色编织绳，长度经测量为1.7米，远超出该机型任何官方配件规格。

手机须长期保持充电状态。经多次穷举测试，该手机的6位锁屏密码无法破解，但系统从未记录任何一次密码错误后的锁定惩罚。手机内部未安装 SIM 卡，信号栏却始终显示满格。

ANO-012 现存放于中国，站点-04，静默通讯室。当前负责专员：？？。

DESCRIPTION

描述

ANO-012 于2010年██月██日首次被记录，回收自██省██市一位独居老人的遗物中。老人的邻居报警称，老人在去世前三天，曾反复向邻居借手机充电器，称自己“在等一个重要电话”。

法医鉴定老人死于心脏骤停，死亡时间是其子从外地赶回家中的前一日。其子向调查人员透露，他正是在那一日接到了母亲用这部手机拨出的电话。

当有人类持续接触 ANO-012 超过约1小时，手机将从待机状态自动亮屏，并向当前距离最近的人类拨出一通电话。通话接通后，听筒中传来的声音为接听者某一位尚在人世的亲属的“临终留言”。

该亲属在通话时处于完全正常的意识状态，正在从事日常活动，对“留言”行为毫不知情。通话内容并非随机截取的日常对话，而是一段对未来临终时刻的真实预演。话语的内容、语气、用词，均与该亲属最终死亡前的状态高度吻合。

经反复验证，通话中的语音无法被录音设备捕捉，仅能被接听者本人听见。通话结束后的瞬间，该亲属将在所处位置突发心脏骤停，抢救无效。若通话中途被强行挂断，亲属也会立即死亡。目前尚未发现任何阻断这一因果链条的方法。

FEAR SOURCE

恐惧来源解析

ANO-012 的恐惧根源来自人类对“临终通话”这一概念的恐惧。接听者听到的留言往往带有极强的私人情感，可能是愧疚、遗憾、责备、期许。

ANCHORING METHOD

锚定方式

专员为 ANO-012 的长期绑定者，每日在静默通讯室内与手机共处8小时。其锚定频率的核心策略为“预设诀别”。

在接受洗脑训练期间，专员被植入一段虚构记忆，使其坚信自己所有在世的亲人皆已在某场事故中离世。因此，当接触手机时，手机无法从专员的认知场中检索到“未完成告别”的恐惧磁场，同频无法持续递进。

APPENDIX

附录：通话记录节选

通话记录001号

接听者：███（时任研究员）

亲属：接听者之父

通话内容概要：“儿子，你上次说工作忙，过年不回来了。没事，我跟你妈说一声。那个……你那个房间的暖气片我前天找人修好了，你妈非要我告诉你，说怕你下次回来冷。啊对了，我给你寄的腊肉你收到没有？你妈说快递单号她找不到了，你收到了说一声，她好放心……”

通话持续11分42秒，期间父亲语气平稳，背景音为电视机播放的戏曲节目。通话结束后第3秒，父亲于家中客厅沙发上去世。尸检显示心脏骤停。

后记：███研究员在事后申请调离 ANO-012 项目，调离申请被批准。

通话记录005号

接听者：███（ANO-012 现任负责专员）

亲属：███之母（虚构记忆对象，实际仍存活）

通话内容概要：这通电话发生在███接受锚定训练的第██天。当时███对亲人的“已故”信念尚未完全固化，手机捕捉到了一次认知缝隙中的微弱恐惧。

“妈妈今天在菜市场看见一件棉袄，红色的，跟你小时候那件特别像。我想买来着，又怕你说我乱花钱……算了，等你过年回来，你自己去试。反正从小到大你穿什么都好看。”

███在通话中始终保持沉默，没有挂断。通话持续4分03秒后自行中断。事后确认，███之母在那一时段正在██市菜市场，精神状态正常，无任何异常感知。通话中断后，███之母未发生心脏骤停。

该事件后，███与手机的同频进入了一种奇特的稳定态。手机不再尝试向她拨打影响现实的通话，而是偶尔响起一段白噪音，像是菜市场的嘈杂声，或电视机的沙沙响。

通话记录012号

接听者：██（清洁人员，未授权接触）

亲属：接听者之女

通话内容概要：清洁人员██在进行 ANO-012 收容室例行打扫时，因疏忽与手机共处超过1小时，触发通话。据██事后口述，接通后她听见5岁女儿的声音，语气天真而平静：

“妈妈，我数到十，你就来接我好不好？一、二、三——”

██女士称，女儿在电话中数到七时停下了，然后笑起来，说：“我数错了，重新数。一、二、三、四……”

██女士在听到第五声时惊恐地挂断了电话。女儿在幼儿园午睡期间突发心脏骤停，抢救无效。幼儿园教师反映，当天上午女儿一直在独自数数，从一数到十，再从一数到十。问她在数什么，她说：“等妈妈来接我的时间。但我一直数错。”

后记：██女士于事故后第7日从站点-04离职。离职面谈中她提到，女儿有先天性智力障碍，5岁仍无法从1数到10。

“那通电话里她数到了五。她这辈子从来没数到过五。我挂断了。我挂断了。”

██女士目前仍在接受心理干预。

CURRENT STATUS

收容现状

ANO-012 现存放于站点-04静默通讯室，房间内仅设一张木桌、一部充电座，以及一把椅子。专员每日独自进入房间，用一块红色绒布反复擦拭手机屏幕。

但站点监控记录显示，专员有时会对着没有接通的手机说话，声音很轻，听不清内容。
