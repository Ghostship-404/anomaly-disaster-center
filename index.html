FILE 001

怪异

人类永远会对未知与超出认知的事物感到恐惧。

恐惧从来不是简单的情绪，它诞生于体液与神经电流的精密交织。正因如此，当恐惧第一次于实验室中被凝炼为切实可触的物质时，整个科学界为之沸腾。

然而这份狂喜并未持续太久。恐惧能够实体化——这个“未知”本身，迅速卷席为全人类的共识。在庞大群体认知的共振之下，深埋于心底的恐惧在一夜之间凝华，化为与人类共栖于地球上的活生生的噩梦。

我们称其为“怪异”。这是唯一能够安全使用的名字，因为若连这个词汇也被恐惧污染，它本身就将凝华为新的“怪异”。

FILE 002

锚定者

恐惧源于未知。倘若人类对某一事物了如指掌，便不会再对其生出恐惧，亦不会催生“怪异”。

正如婴孩不会对哺育自己的母亲产生敌意，训练有素的引导犬不会背叛自己的饲主。锚定者经由严苛的后天洗脑训练，获得了将“怪异”锚定的能力。

他们的存在本身，便足以对抗恐惧的凝华。

人人都有机会成为锚定者，但这个过程因人而异。锚定者需要抛弃许多可能会让自己产生恐惧的东西，比如社会关系、财富、自由、甚至是回忆、梦想、天赋。人无完人，人类只能尽可能去培养一个完美的，对自己的恐惧有所掌控的角色来保护自己。

FILE 003

锚定点

“怪异”收容的核心之地，范围遍布全球各地。

所有锚定者皆诞生于此。每一名锚定者都必须在锚定点中与“怪异”同频，从而理解并化用这些存在所携带的力量。一旦锚定者死去，锚定点必须尽快寻找新的锚定者来与“怪异”同频，以防“怪异”暴动。

然而，有一部分“怪异”无法被常规收容。因此，需要有专员长期驻守，负责与各自对应的“怪异”持续维持同频绑定。

还有一部分“怪异”正处于缓慢的彻底消除进程中，无需与锚定者进行同频。

MATERIAL

恐惧

纯黑的固体结晶，这便是恐惧的物质形态。人类若与之接触过久，便会在无形中进入同频，最终使这块结晶本身凝华为活着的“怪异”，而人类也会被“怪异”俘获，成为它的一部分。

MECHANISM

同频

同频，是指人类的认知与“怪异”的存在频率发生共振的现象。

恐惧之所以能凝华为实体，是因为人类的大脑在处理恐惧时，会释放特定频率的神经电信号。当这个频率与“怪异”的本质频率重合时，双方就会进入同频状态。简单来说，恐惧是同频的钥匙，而同频是恐惧成为实体的桥梁。

非锚定者进入同频状态会根据“怪异”的情况被俘获，需要专业的锚定者进行疏导才能恢复。

认知残留

怪异的部分思维模式或感知方式残留在锚定者脑海中，表现为怪癖、幻觉或无法解释的直觉。

频率偏移

锚定者自身的认知频率逐渐向所绑定的怪异靠近，最终可能无法再与正常人类共情。

共毁风险

若锚定者的精神防线崩溃，双方频率将发生逆转，怪异反客为主。锚定者将被抹除或转化为新的威胁，被称为逆锚定。

戒断反应

长期绑定某一怪异的锚定者，若被调离或终止绑定，可能出现严重的精神空缺症状，仿佛失去了一部分自己。

CONTAINMENT

收容

收容“怪异”，本质是与恐惧赛跑。

物理收容是最直接的手段，把已经凝华的“怪异”封闭进特制区域，让锚定者进行同频以控制“怪异”活动。

概念弱化才是釜底抽薪。既然恐惧源于未知，那就把未知变成常识。让人类相信那只是自然现象，用娱乐把它塑造成笑话，用教育让下一代不再害怕。当全人类都不再恐惧某个概念时，对应的怪异便会失去养分，逐渐瓦解。这个过程很慢，慢到需要一代人忘记恐惧。

还有一些怪异无法被物理触碰，也无法被公开讨论，因为光是知道它的存在就足以触发同频。对它们，只能使用逻辑收容：让它困在一个永远无法抵达下一秒的时间闭环里，或者让所有参与收容的人集体忘记自己做了什么。
